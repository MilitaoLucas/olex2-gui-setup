Functions in Olex 1

This tutorial is showing what basic functions are in Olex.

 

  • Load the sucrose structure>On the main menu, Select File > Open File or>Click on   in the toolbar above OR type 'reap'

 

  • Left-Click (and hold) on the background of the main window. Moving the mouse will rotate the structure

 

  • Alt + Left-Click (and hold) on the background of the main window. Moving the mouse will move the structure

 

  • Left-Click on an atom. The atom will be selected and change colour. Now select another atom. This atom will be added to the current selection and change colour also.

 

  • Shift + Left-Click (and hold) all the atoms and Q-peaks will be selected inside the square

 

Proceed if you are able to use those functions
