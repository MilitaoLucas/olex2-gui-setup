To display the labels of the  atoms in the structure, type labels, or click on this icon  above.

The model can be manipulated using the mouse:

  • Left-click (and hold) on the background of the main window. Moving the mouse while holding the left mouse button will move the structure
  • Left-click on an atom. The atom will be selected and change colour. Now select another atom. This atom will be added to the current selection and change colour also.
